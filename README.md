# EnsoAutoBot-NTE
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/airdropbomb/EnsoAutoBot.git && cd EnsoAutoBot
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Enter Your PrivateKey And Zealy ID

   ```bash
   nano accounts.txt
   ```
4. If You Want Use Proxy
   ```bash
   nano proxy.txt
   ```
## Usage

- **Start the bot**:

  ```bash
  npm start
  ```
